# 泰享购 - 微信小程序商城

一个功能完整的微信小程序商城项目，使用模拟数据，无需后台支持。

## 功能特性

### 核心功能
- ✅ 首页展示（轮播图、分类导航、商品推荐）
- ✅ 商品分类浏览
- ✅ 商品搜索和筛选
- ✅ 商品详情页（规格选择、数量选择）
- ✅ 购物车管理（增删改查、全选/反选）
- ✅ 订单管理（下单、支付、取消、确认收货）
- ✅ 个人中心（订单统计、功能菜单）

### 页面结构
```
pages/
├── index/          # 首页
├── category/       # 分类页
├── goods-list/     # 商品列表页
├── goods-detail/   # 商品详情页
├── cart/           # 购物车页
├── order/          # 订单提交页
├── order-detail/   # 订单详情页
├── order-list/     # 订单列表页
└── profile/        # 个人中心页
```

## 技术栈

- 微信小程序原生开发
- 本地存储（wx.setStorageSync/wx.getStorageSync）
- 模拟数据（data/mockData.js）

## 快速开始

### 1. 导入项目

1. 打开微信开发者工具
2. 选择"导入项目"
3. 选择项目目录
4. 填写 AppID（可以使用测试号）

### 2. 运行项目

项目导入后即可直接运行，所有数据都是模拟数据，无需配置后台。

### 3. 数据结构

所有数据存储在 `data/mockData.js` 文件中，包括：
- 轮播图数据
- 分类数据
- 商品数据
- 推荐商品列表

## 功能说明

### 购物流程

1. **浏览商品**
   - 首页查看推荐商品
   - 分类页按分类浏览
   - 搜索页搜索商品

2. **查看详情**
   - 点击商品进入详情页
   - 选择商品规格（如有）
   - 选择购买数量

3. **加入购物车**
   - 在详情页加入购物车
   - 在购物车页管理商品

4. **下单支付**
   - 选择收货地址（模拟）
   - 提交订单
   - 模拟支付流程

5. **订单管理**
   - 查看订单列表
   - 查看订单详情
   - 取消订单/确认收货

### 数据存储

项目使用微信小程序的本地存储功能：
- `cart`: 购物车数据
- `orders`: 订单数据
- `userInfo`: 用户信息
- `defaultAddress`: 默认收货地址

## 注意事项

1. **图片资源**
   - 项目中使用的是 Unsplash 的图片链接
   - 实际项目中需要替换为自己的图片资源
   - 建议使用 CDN 或本地图片

2. **TabBar 图标**
   - 项目配置了 TabBar，但需要添加对应的图标文件
   - 图标路径：`images/home.png`, `images/category.png` 等
   - 可以使用图标字体或图片资源

3. **支付功能**
   - 当前为模拟支付，实际项目需要接入微信支付
   - 需要配置支付参数和回调处理

4. **数据持久化**
   - 当前使用本地存储，数据仅保存在本地
   - 实际项目需要对接后台 API
